## Summary

candle-gen is a Rust-based image generation framework built on top of the Candle ML library (Hugging Face's Rust ML framework). The repository implements diffusion model pipelines for text-to-image generation. The primary sub-project, candle-gen-chroma, implements the Chroma diffusion model with support for multiple model variants (base, HD, flash) and configurable noise schedules including beta-distributed sigma schedules and linear sigma schedules used in denoising diffusion probabilistic models (DDPMs). The framework handles model loading, configuration management, and inference pipeline orchestration.
