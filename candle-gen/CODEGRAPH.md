## Summary

candle-gen is a Rust-based image generation framework that provides a backend implementation using the Candle ML framework (HuggingFace's Rust ML library) for running Stable Diffusion XL (SDXL) inference. It exposes a pipeline for text-to-image generation, including text encoding via CLIP, latent diffusion, and VAE decoding, while integrating with a gen-core abstraction layer that defines common generation interfaces like ModelDescriptor, GenerationRequest, and Progress reporting.
