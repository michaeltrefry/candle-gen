## Summary

candle-gen is a Rust-based image generation framework built on top of the Candle ML library. The candle-gen-chroma sub-project specifically implements the Chroma diffusion model pipeline, supporting multiple model variants (base, HD, flash) with configurable noise schedules including beta-distributed sigma schedules and linear sigma schedules for denoising diffusion probabilistic models (DDPMs). It provides model loading, configuration, and inference pipeline logic for text-to-image generation.
