## Summary

candle-gen is a Rust project that implements image generation pipelines using the FLUX diffusion model family (including FLUX.1-schnell, FLUX.1-dev, and FLUX2/Klein variants), built on top of the Candle ML framework. It provides model loading, configuration, scheduling, and pipeline execution for text-to-image generation, exposing a plugin/registry system where different model variants register themselves as Generator implementations.
