## Summary

candle-gen is a Rust-based image generation framework built on the Candle ML library, implementing diffusion model pipelines. The candle-gen-chroma subproject specifically implements the Chroma diffusion model with multiple variants (base, flash, HD), including beta-schedule noise sigma computation, classifier-free guidance configuration, and image generation pipelines using packed noise tensors.
