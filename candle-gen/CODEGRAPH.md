## Summary

candle-gen is a Rust project that provides an image generation pipeline built on top of the FLUX diffusion model (via the candle ML framework). It implements two model variants (Schnell and Dev), handles model loading from safetensor files via memory-mapped var builders, runs the denoising diffusion loop, and exposes a Generator interface with model descriptors for registration and resolution by a higher-level gen-core registry.
