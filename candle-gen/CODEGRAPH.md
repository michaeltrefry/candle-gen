## Summary

candle-gen is a Rust-based image generation framework built on the Candle ML library, implementing the Chroma diffusion model with three variants (Base, HD, Flash). It provides a pipeline for flow-matching-based image generation using configurable noise schedules (beta or linear), transformer architecture configuration, and a registry-based loading system for different model weights.
