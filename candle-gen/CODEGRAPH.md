## Summary

candle-gen is a Rust-based image generation framework built on top of the Candle ML library, implementing diffusion model pipelines. The candle-gen-chroma sub-project specifically implements the Chroma image generation model in three variants (base, flash, HD), including noise scheduling via beta distribution sigma schedules, model configuration, and pipeline orchestration for text-to-image generation.
