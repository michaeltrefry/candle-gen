## Summary

candle-gen is a Rust-based image generation framework that implements diffusion model inference using the Candle ML framework. The candle-gen-chroma subproject specifically provides support for Chroma diffusion models (in 'flash' and 'hd' variants), including beta-schedule noise sigma computation, model configuration, and a pipeline that wires together samplers and schedulers for text-to-image generation.
