## Summary

candle-gen is a Rust-based image generation framework built on top of the Candle ML library. The candle-gen-chroma subproject implements a diffusion-based image generation pipeline supporting multiple model variants (base, flash, HD) with configurable noise schedules, including a beta distribution-based sigma schedule. It provides model loading, pipeline execution, and configuration management for text-to-image generation using flow-matching or diffusion inference.
