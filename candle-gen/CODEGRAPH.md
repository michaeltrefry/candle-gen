## Summary

candle-gen is a Rust-based image generation framework built on the Candle ML library, implementing diffusion model pipelines (specifically Chroma variants: base, HD, and flash) with noise scheduling, beta distribution-based sigma schedules, and a plugin/registry system for loading different model configurations. It provides the core infrastructure for running text-to-image diffusion models with configurable sampling schedules.
