## Summary

candle-gen is a Rust-based image generation framework built on top of the Candle ML library. The candle-gen-chroma sub-project implements support for the Chroma diffusion model family (base, flash, and HD variants), providing noise scheduling via beta distribution-based sigma schedules, model loading/configuration, and an image generation pipeline using packed noise tensors.
